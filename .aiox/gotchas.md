## API

### Excluir um job em processamento ('processing' ou 'pending...

**[HIGH]**

**Wrong:**
```typescript
Permitir a exclusão de qualquer job via API sem verificar o status.
```

**Right:**
```typescript
Validar se o status do job é 'completed', 'failed' ou 'cancelled' antes de executar o hardDelete.
```

**Reason:** Excluir um job em processamento ('processing' ou 'pending') pode causar inconsistências na fila do Bull e erros de processamento em segundo plano.

**Severity:** High

**Discovered:** 1.3 (2026-04-21)

**Related Files:** backend/src/controllers/jobController.ts

## Database

### Sem os filtros, a paginação no frontend exibirá um número...

**[MEDIUM]**

**Wrong:**
```typescript
Realizar a contagem total de jobs sem aplicar os mesmos filtros da listagem.
```

**Right:**
```typescript
Garantir que a query de count() receba o mesmo objeto JobFilter que a query de listagem.
```

**Reason:** Sem os filtros, a paginação no frontend exibirá um número incorreto de páginas e resultados totais quando um filtro for aplicado.

**Severity:** Medium

**Discovered:** 1.3 (2026-04-21)

**Related Files:** backend/src/repositories/jobRepository.ts
