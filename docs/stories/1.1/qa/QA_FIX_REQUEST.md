## Issues to Fix

### 1. [CRITICAL] Incomplete Progress Calculation (Stuck at <100%)

**Issue ID:** FIX-1.1-001

**Location:** `backend/src/controllers/statusController.ts`

**Problem:**
The progress calculation only sums `completedCount + stats.failed`. If an email is "bounced" or "skipped" (idempotency), it is not counted, preventing the bar from reaching 100%.

```typescript
const progress = total > 0 ? Math.round(((completedCount + stats.failed) / total) * 100) : 0;
```

**Expected:**
Sum all terminal states including `bounced`.

```typescript
const totalProcessed = stats.sent + stats.failed + stats.bounced;
const progress = total > 0 ? Math.round((totalProcessed / total) * 100) : 0;
```

**Verification:**
- [x] Simulate a job with 10 emails where 1 is bounced.
- [x] Verify progress reaches 100% when all 10 are processed.

**Status:** [x] Fixed
