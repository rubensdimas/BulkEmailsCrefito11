## 📋 User Story

**Como** desenvolvedor utilizando o Opencode CLI,
**Quero** que as ferramentas, hooks e agentes do aiox-core funcionem nativamente,
**Para** ter a mesma experiência de orquestração e automação do Gemini CLI, adaptando a pasta `@.gemini/hooks/` para o contexto do Opencode.

---

## 🎯 Objective

Configurar o Opencode para suportar o ecossistema AIOX, garantindo que a ativação de agentes, execução de comandos e orquestração de sessões funcionem via hooks adaptados.

## 🎯 Acceptance Criteria

```gherkin
GIVEN o Opencode CLI está instalado
WHEN eu inicio uma sessão no Opencode
THEN o hook de session-start deve carregar o contexto do aiox-core

GIVEN eu invoco um agente via prefixo no Opencode (ex: /aiox-master)
WHEN o agente é ativado
THEN ele deve exibir o greeting oficial do Orion com contexto de projeto e sessão

GIVEN eu executo um comando AIOX no Opencode
WHEN o comando termina
THEN o estado da sessão deve ser updated em .aiox-core/.session/current-session.json
```
