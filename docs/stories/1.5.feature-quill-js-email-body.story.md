## 📋 User Story

**Como** usuário do sistema de envio de emails em massa,
**Quero** utilizar um editor de texto rico (Quill JS) para compor o corpo do email,
**Para** ter mais flexibilidade na formatação do conteúdo (negrito, itálico, listas, links) e gerarHTML formatado para o email.

---

## 🎯 Objective

Substituir o componente textarea simples atual, utilizado para digitar o corpo do email, por um editor de texto rico utilizando a biblioteca Quill JS. O conteúdo digitado deve ser convertido para formato HTML para ser enviado junto com o email.

## 🎯 Acceptance Criteria

```gherkin
GIVEN usuário está na página de compose de email
WHEN usuário digita no editor Quill e adiciona formatação (negrito, lista)
THEN o conteúdo é armazenado como HTML
AND ao enviar o email, o corpo contém as tags HTML correspondentes

GIVEN usuário limpar o conteúdo do editor
WHEN clicar no botão de limpar ou remover o texto
THEN o campo deve ficar vazio (não deve enviar código HTML vazio)

GIVEN usuário está em dispositivo móvel
WHEN acessa a página de compose
THEN o editor Quill deve ser responsivo e utilizável
```
